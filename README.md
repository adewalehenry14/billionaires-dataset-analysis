# Global Billionaires Dataset

## Overview
This dataset contains information on the world's 
billionaires used for wealth and trend analysis.

## Contents
- Billionaire names and net worth
- Industry/sector breakdown
- Geographic distribution
- Wealth growth trends

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Use Cases
- Wealth distribution analysis
- Industry comparison
- Geographic insights
- Year-on-year trend analysis
